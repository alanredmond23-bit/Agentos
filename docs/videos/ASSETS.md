## Asset Delivery

### File Organization

```
assets/
  video-01/
    graphics/
    lower-thirds/
  video-02/
    graphics/
    troubleshooting-cards/
  ...
  shared/
    branding/
    backgrounds/
    templates/
  b-roll/
  audio/
```

### Delivery Format

- Graphics: PNG (raster), SVG (vector)
- Animations: MP4 (H.264), GIF for simple loops
- Audio: MP3 (compressed), WAV (uncompressed)
- Templates: PSD, Figma source files

### Naming Convention

```
[video-number]-[category]-[description].[ext]

Examples:
01-graphic-architecture-diagram.png
01-lt-security.png
shared-logo-dark.svg
broll-typing-01.mp4
```
